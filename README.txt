----- Sistema de controle de Jornada de Trabalho ----- 

Ferramentas Utilizadas:
1 - Banco de Dados PostgresQl 12
2 - NodeJs v12.16.1 (Guia de instalação - https://nodejs.org/pt-br/download/package-manager/)
3 - TomCat 8.5 (Guia de instalação)
4 - Eclipse
5 - Java 8
6 - Visual Studio Code
7 - Insominia

Instalação do backend:

1 - Iniciar o Eclipse.
2 - File > Import > Maven > Existing Maven Project
3 - Browse (Procurar a pasta scjdt-projeto) > Selecionar o Projeto > Next > Finish
4 - Fazer a instalação do Tomcat 8.5 conforme
5 - Adicionar ao servidor o Projeto
6 - Rodar o servidor

Instalação do frontend:

1 - Com o NodeJs instalado ir na pasta raiz do fronted pelo terminal e digitar o comando "npm i"
2 - Após instalado as dependências usar o comando "npm start"
